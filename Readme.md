# Mysql Docker

## Description

- Runs using docker-compose
- Maps the /data and /conf folders
- Don't keep the /data content
- mysqld.cnf stored to set the software requisites.
- Maps the 3306 port to localhost
- Mysql JDBC connector stored

## Requirements

- Docker
- docker-compose
- terminal :)

## How-to

```sh
	docker-compose up -d
```
